# Laravel Portfolio Web Application

## Project Description
This project is a simple portfolio-style web application developed using the Laravel framework as part of Laboratory 2.  
The application demonstrates the basic use of Laravel routing, controllers, and Blade templates by creating multiple web pages connected through controllers and routes.

---

## Pages Included

### 1. Main Portfolio Page
**Route:** `/`  
This page serves as the main portfolio page. It introduces the student and provides navigation links to the other pages.

**Content Example:**
- Welcome message
- Short self-introduction
- Navigation links to About and Hobbies pages

---

### 2. About Me Page
**Route:** `/about`  
This page contains basic information about the student.

**Content Example:**
- Short background description
- Academic or personal interests
- Link back to the main portfolio page

---
